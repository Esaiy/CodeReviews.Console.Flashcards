# Flashcard App

## Setup

Set the value of `ConnectionString` in the `App.config` to your SQL Server before running the program.

## Features

- Manage stack and flashcard.
- Do a study session on a stack of flashcard.
- View the results of past study sessions.
- View the yearly report of study sessions.
